
# ansvgx
## 介绍 / Intro

基于adobe animate导出的svg图片，封装的一些组件和工具方法 。


## 安装 / Install
```
npm i ansvgx
```

### import 整个包 
```javascript
import * as svgx from 'ansvgx'
```

### import 单个组件

```javascript
import {HoverButton} from 'ansvgx'
```

## 使用 / Use  

```javascript
const widget = new svgx.HoverButton(el);    
```
